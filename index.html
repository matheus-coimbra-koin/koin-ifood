POST /v1/webhooks/ifood_tem_que_criar_uma_api/credit-limit-acceptance
Koin -> Ifood: JOB - Webhook recálculo de limite de crédito
Endpoint utilizado pelo fluxo interno da Koin para enviar ao iFood o estado completo do limite de crédito recalculado. O payload contém todos os dados consolidados do limite (partner_limit + limites operacionais), permitindo que o iFood atualize sua base e siga com o fluxo de notificação ao cliente. O processamento é assíncrono, com envio via integração (webhook/SQS).
